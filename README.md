## Git Grapher

*A script to fill your Github contributions graph so that you can flaunt it
to your friends*

### How to
```bash
git clone https://github.com/UtkarshMe/GitGrapher
cd GitGrapher
rm -rf .git
./gitgrapher
```
Create a repo on github and push this code to it.
*It's that simple!*
